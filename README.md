# 🚀 Clash 分流规则配置（高质量整合版）

一套基于 ACL4SSR + blackmatrix7 深度整合的 Clash 分流配置，适用于日常科学上网、流媒体解锁、AI服务访问、广告拦截等多场景。

> 适合：Clash / OpenClash / Meta / Stash 用户

---

## ✨ 特点

* ✅ 精细化分流

  * AI服务（OpenAI / Claude / Gemini）独立分组
  * 流媒体（Netflix / Disney+ / HBO / YouTube 等）独立策略
  * 国内 / 国外流量自动分流

* ✅ 广告拦截 + 隐私保护

  * 集成 ACL4SSR + EasyList + 隐私规则
  * 支持应用净化 / 广告拦截 / 防追踪

* ✅ DNS 优化（重点优化）

  * 国内 DNS + 国外 DoH 双体系
  * 防污染 / 防泄漏 / 避免回环解析
  * fake-ip 精细过滤（银行 / 微信 / 系统服务）

* ✅ **支持 IPv6 节点连接（重要）**

  * 已开启 `ipv6: true`，支持节点走 IPv6 出口
  * DNS 层禁用 IPv6 解析，避免 AAAA 污染
  * 兼顾 IPv6 可用性与稳定性

* ✅ 多策略节点调度

  * 自动选择（延迟测试）
  * 故障转移（Fallback）
  * 负载均衡（Load Balance）

* ✅ 地区节点自动归类

  * 🇭🇰 香港 / 🇯🇵 日本 / 🇺🇸 美国 / 🇸🇬 新加坡 等自动分组

---

## 📦 配置结构说明

### 1️⃣ DNS 设计（核心亮点）

* 国内解析：223.5.5.5 / 119.29.29.29
* 国外解析：Cloudflare / Google DoH
* 避免 DNS 污染、解析绕路

📌 特别处理：

* 银行 / 微信 / 支付类 → 直连
* 系统更新 → 避免 fake-ip 干扰

📌 IPv6 策略：

* DNS 禁用 IPv6（防污染）
* 节点连接允许 IPv6（提升连通性）

---

### 2️⃣ 分流逻辑

优先级：

广告/隐私 → AI服务 → 流媒体 → 国外流量 → 国内流量 → 兜底

示例：

* OpenAI → AI服务
* Netflix → Netflix分组
* 中国IP → 国内流量
* 其他 → 漏网之鱼

---

### 3️⃣ AI 服务专属优化

特点：

* 独立策略组
* 可手动指定节点（建议 US / SG / JP）
* 避免污染或错误分流

---

### 4️⃣ 流媒体解锁支持

支持：

* Netflix
* Disney+
* HBO
* Hulu
* YouTube
* PrimeVideo
* AppleTV+

每个平台独立策略组，可单独切换节点。

---

### 5️⃣ 广告 & 隐私防护

功能：

* 拦截网页广告
* 屏蔽应用广告
* 防止数据追踪

---

### 6️⃣ 节点策略说明

| 策略   | 说明       |
| ---- | -------- |
| 自动选择 | 延迟最低     |
| 故障转移 | 自动切换可用节点 |
| 负载均衡 | 分流负载     |
| 手动选择 | 自定义      |

## ⚙️ 推荐设置

* 模式：Rule
* DNS：开启
* IPv6：

  * 连接：开启（推荐）
  * DNS解析：关闭

## 🧠 设计思路

这套配置重点解决：

1. DNS污染导致无法访问
2. AI服务异常
3. 流媒体解锁失败
4. 国内服务误走代理
5. 广告过多 & 隐私泄露
6. IPv6 环境下连接不稳定问题
